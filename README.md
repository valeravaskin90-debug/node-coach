# Node Coach — dual game website

Общий GitHub Pages-сайт для двух продуктов:

- Node Coach CS
- Node Coach Dota

## Главная функция

Верхний переключатель меняет:

- цветовую тему;
- описание продукта;
- метрики в демонстрационном окне;
- приоритеты тренера;
- набор функций;
- текст и ссылку загрузки.

CS использует мятно-бирюзовую тему. Dota — красно-бордовую.

## Публикация

Запусти:

```text
PUBLISH_TO_GITHUB.cmd
```

Для последующих обновлений:

```text
UPDATE_GITHUB_SITE.cmd
```

## Адрес

Обычный project-site:

```text
https://GITHUB_LOGIN.github.io/node-coach/
```

## Загрузки

Пока кнопки ведут на страницы статуса beta:

- `download-cs.html`
- `download-dota.html`

После появления готовых Windows-сборок их можно заменить ссылками на GitHub Releases.
